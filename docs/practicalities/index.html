Practicalities

Presentations and slides

Please note that you can only use the main laptop (running on Wondows 11) the organizers make available in each room.

File format

In person presentations: Please prepare a file of your slides in PDF or PPT format.

Distant presentations: Please prepare an audivisual recording of your presentation.

Upload

Please upload your file to the cloud folder of the conference (information provided by email) by Monday 8th December. Make sure to add your file to your session’s subfolder and use a clear file name (including main presenter’s surname).

Duration

Please make sure to stick to your allotted time.

Long papers: 15 minutes + 7.5 minutes Q&A

Short papers: 10 minutes + 5 minutes Q&A

Lightning talk: 3 minutes

The day of the conference, please make sure to arrive to your session 10 minutes before schedule to ensure smooth set-up and a prompt start.

Use of Webex (for distant presenters)

Please ensure that you have downloaded and installed Webex on your computer in advance.

Take the time to familiarize yourself with the tool by creating a Webex test meeting. Also, please practice sharing your screen (normally you will not have to do this, but please be ready just in case). If Webex has been newly installed, you might be asked to grant permission for screen sharing. This step requires a computer restart, so please plan accordingly.

If you wish to have a little training session before the conference takes place, please send an email to Vanessa Napolitano.

Posters

Format

Posters are to be A1 format (59,4 cm x 84,1 cm).

Set up

Set up will take place on the same day as the poster session (11th December) after the lunch break in the catering area (level 3, Maison du Savoir, Belval Campus). If possible, we ask for posters to please be left in place until the end of conference to allow all attendees the opportunity to view them.

Where to print in the area

Mister Copy. Located at 9b Bd Prince Henri, 1724 Ville-Haute Luxembourg

Offers A1 format printing from PDF files provided on-site. Orders can be placed via email. Upon receipt of prior payment, posters will be ready for client pickup on-site. Please allow 2-3 days for processing.

Multiprint. Located at 233 Rue François Ponsin, 57390 Audun-Le-Tiche.

Offers A1 format printing from PDF files. Orders are accepted via email only. Upon receipt of prior payment, posters will be ready for client pickup on-site. Please allow 1-2 days for processing.
